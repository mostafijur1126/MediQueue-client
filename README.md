## MediQueue 🎓

MediQueue is a modern tutor booking web application where students can easily register, log in, browse tutors, and book online learning sessions based on subject preferences and tutor availability. The platform streamlines the entire booking process with digital session management and organized scheduling.

🌐 Live Website link: https://medi-queue-client.vercel.app

## Features

- Secure user authentication and authorization system
- Browse tutors by subject, availability, and teaching mode
- Book tutoring sessions based on available time slots
- Automatic digital session token generation after booking
- Prevents duplicate or conflicting booking schedules
- Fully responsive modern UI with dark and light mode support
- Easy session management for students and tutors
- Supports online, offline, and hybrid teaching modes

## Project Purpose

MediQueue was developed to simplify the tutor booking experience by:

Reducing manual scheduling problems
Preventing slot conflicts
Providing a smooth learning session management system
Helping students quickly connect with suitable tutors
